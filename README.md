# Уютное Кафе

## Структура проекта

- `landing.html` - Главная страница (лендинг)
- `index.html` - Главная страница приложения
- `lunch.html` - Страница для сборки ланча
- `checkout.html` - Страница оформления заказа
- `orders.html` - Страница истории заказов
- `dishes.js` - Загрузка блюд из API
- `order.js` - Управление состоянием заказа
- `submit_order.js` - Отправка заказа на сервер
- `orders_history.js` - Управление историей заказов
- `validation.js` - Валидация формы заказа

## Функциональность

✅ **Просмотр и выбор блюд** - Категории: супы, основные блюда, салаты, напитки, десерты  
✅ **Сборка ланча** - Интерактивный выбор блюд с проверкой комбо  
✅ **Оформление заказа** - Форма с валидацией и отправкой на сервер  
✅ **История заказов** - Просмотр, редактирование и удаление заказов  
✅ **Адаптивный дизайн** - Работает на всех устройствах  

## Технологии

- **HTML5** - Структура страниц
- **CSS3** - Стилизация и адаптивность
- **JavaScript (ES6+)** - Логика приложения
- **Fetch API** - Работа с внешним API
- **LocalStorage** - Сохранение состояния заказа

## API

Приложение работает с API: `https://edu.std-900.ist.mospolytech.ru/labs/api/`

Эндпоинты:
- `GET /dishes` - Список блюд
- `GET /orders` - Список заказов пользователя
- `POST /orders` - Создание нового заказа
- `PUT /orders/{id}` - Редактирование заказа
- `DELETE /orders/{id}` - Удаление заказа

API ключ передается в query параметре `api_key`.
